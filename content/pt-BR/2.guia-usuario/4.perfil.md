# Perfil

A página de perfil (`/profile`) permite gerenciar informações pessoais, visualizar estatísticas e alterar a senha.

## Cartão do Perfil (Lateral Esquerda)

### Avatar e Informações

- Avatar circular com a inicial do nome (fundo laranja)
- Botão de câmera para upload de foto de perfil
- Nome e e-mail do usuário
- Badge de papel: **Admin** (azul) ou **Agente** (verde)
- Data de ingresso
- Número do ramal
- Tag de status atual com cores ao vivo

### Estatísticas (Somente Agentes)

Cartão com métricas calculadas automaticamente:

- **Taxa de Atendimento** - Barra de progresso com percentual
- **Chamadas Atendidas** vs **Total de Chamadas**
- **Chamadas Hoje** (laranja)
- **Chamadas Perdidas** (vermelho)
- **Duração Média** - Formatado como Xm Ys

## Informações Pessoais (Lado Direito)

### Modo de Visualização

Exibe nome, e-mail e papel (somente leitura).

### Modo de Edição

Clique em "Editar" para alternar:
- Campo de nome (obrigatório)
- Campo de e-mail (obrigatório, com validação)
- Papel exibido como somente leitura
- Validação em tempo real com mensagens de erro
- Botões Salvar/Cancelar com estados de carregamento
- Notificação toast ao salvar

## Alterar Senha

Seção com três campos:

1. **Senha Atual** - Obrigatório
2. **Nova Senha** - Mínimo 8 caracteres
3. **Confirmar Senha** - Deve ser igual à nova senha

### Requisitos da Senha

- Mínimo de 8 caracteres
- Pelo menos uma letra maiúscula
- Pelo menos uma letra minúscula
- Pelo menos um número

### Recursos

- Botão de alternar visibilidade (ícone de olho)
- Indicador de força da senha
- Tooltip com os requisitos
- Formulário limpa automaticamente após sucesso
- Notificação toast de confirmação

## Zona de Perigo

Cartão com borda vermelha na parte inferior:

### Desativar Conta

- Botão "Desativar Conta" (vermelho)
- Diálogo de confirmação detalhado com lista de consequências:
  - Desconectar de todas as chamadas
  - Remover acesso ao sistema
  - Dados retidos mas inacessíveis
- Ao confirmar: faz logout e redireciona para login
- Lembrete para contatar o administrador

::callout{type="warning" title="Desativar Conta"}
A desativação não exclui os dados do agente. Um administrador pode reativar a conta se necessário.
::
